## Objetivo del Proyecto

El propósito de este portafolio es demostrar, a través del propio código, el dominio de las tecnologías base de la web. Está diseñado para:

- Servir como prueba técnica viva de habilidades en frontend.
- Centralizar proyectos, experiencia y contenido multimedia en un solo hub digital.
- Proyectar una identidad profesional única con estética futurista (cyberpunk).
- Establecer un canal de contacto directo con empleadores o clientes.

---

## 🛠️ Stack Tecnológico

| Tecnología | Uso en el Proyecto |
| :--- | :--- |
| **HTML5** | Estructura semántica del contenido y base para el SEO. |
| **CSS3** | Sistema de diseño completo (variables, transiciones, animaciones) para la estética ciberpunk y la adaptación de modo Dark/Light al 100% de los elementos. |
| **JavaScript (Vanilla)** | Lógica del fondo Matrix (Canvas), filtro dinámico de proyectos, toggle de modo, validación de formularios y envío de correos. |
| **Bootstrap 5.3** | Sistema de grid responsivo, componentes de navbar y tarjetas, y utilidades de espaciado. |
| **Font Awesome 6** | Iconografía vectorial escalable que se adapta a los cambios de color del modo oscuro/claro. |
| **EmailJS** | Servicio backend para envío de correos electrónicos directamente desde el frontend, sin necesidad de servidor. |

## Características Principales

- **Modo oscuro**  
  Cambia todos los elementos (textos, fondos, botones, iconos, bordes) manteniendo un contraste perfecto y una estética vibrante en ambos modos. La preferencia se guarda en el navegador.

- **Fondo Matrix Translúcido:**  
  Canvas animado con caracteres estilo "Matrix" en posición fija, con opacidad baja y color adaptable (verde neón / verde oscuro) que no interfiere con la legibilidad.

- **Filtro Dinámico de Proyectos:**  
  Sección de portafolio con 9 proyectos organizados en 3 categorías (`Web`, `App`, `Diseño`). Botones interactivos permiten filtrar el contenido al instante.
